# ТралСамара — Сайт аренды спецтехники и услуг

Демо-сайт для компании по аренде спецтехники и грузоперевозкам в Самаре.

## Содержание

- **Главная** (`index.html`) — лендинг с каталогом техники, калькулятором, услугами
- **Аренда техники** (`catalog.html`) — каталог спецтехники с фильтрацией
- **Услуги** (`services.html`) — доставка инертных, снос зданий, вывоз мусора
- **Политика конфиденциальности** (`privacy.html`) — 152-ФЗ

## Размещение на GitHub Pages

Сайт опубликован: [https://sasha131080-cloud.github.io/tralsamara/](https://sasha131080-cloud.github.io/tralsamara/)

## Соответствие законодательству РФ

- 152-ФЗ «О персональных данных»: согласие на обработку ПДн, политика конфиденциальности
- Ст. 437 ГК РФ: цены указаны как ознакомительные, не являются публичной офертой
- Запрещена передача ПДн третьим лицам без согласия

## Настройка формы

### PHP (хостинг)
1. В `send.php` замените email на ваш
2. Загрузите все файлы на PHP-хостинг
